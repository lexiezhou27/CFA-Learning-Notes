# Common Measures of Return

---

## 1️⃣ Gross Return 总收益率
**Gross return** refers to the total return on a security portfolio before deducting fees for the management and administration of the investment account. 
It measures the performance of the investment itself before the cost of managing the account is considered.

## 2️⃣ Net Return 净收益率
**Net return** refers to the return after the management and administration fees have been deducted.
* **Key Note:** Commissions on trades and other costs necessary to generate the investment returns are typically deducted in *both* gross and net return measures.

---

## 3️⃣ Other Measures of Return

### 1. Pretax Nominal Return 税前名义收益率
The return before paying taxes. This includes dividend income, interest income, and capital gains.

### 2. After-tax Nominal Return 税后名义收益率
The return after the tax liability is deducted.

### 3. Real Return 实际收益率 (Adjusted for Inflation)
Real return measures the increase in an investor's purchasing power. 
* **Approximate Formula:** $$\text{Real Return} \approx \text{Nominal Return} - \text{Inflation}$$
* **Exact Formula:**
  $$(1 + \text{Real Return}) = \frac{1 + \text{Nominal Risk-free Rate} + \text{Risk Premium}}{1 + \text{Inflation Premium}}$$

### 4. Leveraged Return 杠杆收益率
A return to an investor that is a multiple of the return on the underlying asset. 
$$r_L = \frac{r(V_0 + V_B) - r_B V_B}{V_0}$$
* **Note:** $V_0$ is the cash investment, $V_B$ is the borrowed amount, and $r_B$ is the interest rate on borrowed funds.
